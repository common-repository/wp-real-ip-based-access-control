WP Real IP-based Access Control

== Description ==

[WP Real IP-based Access Control](http://www.hitoy.org/wp-real-ip-based-access-control.html) is a Plugin Specifically Designed for Website use CDN service, with this WP Real IP-based Access Control, all of your comments users's ip address are their original IP address instead of your CDN notes IP.

With This Plugin you can not only View your site visitors's real IP, But also you can Control the access based on IP.

You have three ways to control permissions:
	1. Completely closed.
	2. Prohibit Access.
	3. Prohibit Comments.

If you decide to open the access control feature, you need to fill the ip address to the Text field in the setting->WP Real IP-based ACL.



== Installation ==

1. Upload `WP Real IP-based Access Control` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
